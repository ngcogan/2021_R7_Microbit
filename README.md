Go to https://python.microbit.org/ and paste the code.
Method 1:
- Download the file
- Plug in the Microbit into your USB port
- Copy the file to your Microbit

Method 2:
- Click Connect
- Select BBC micro:bit
- Click Connect
- Click Flash
